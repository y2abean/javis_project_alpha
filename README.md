# NEURON (뉴런) - 프로젝트 알파

이 디렉토리는 NEURON 챗봇의 핵심 소스 코드를 포함하고 있습니다.

## 구성 요소

- **Core**: `chatbot.py` (Gemini 연동, 로직)
- **Server**: `server.py` (Flask API)
- **Frontend**: `vite-react-app/` (React 웹 인터페이스)

## 실행

1. **백엔드**: `py server.py`
2. **프론트엔드**: `cd vite-react-app` -> `npm run dev`

자세한 내용은 상위 디렉토리의 `README.md`를 참고하세요.
